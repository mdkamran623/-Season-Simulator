# 🍂 Season Simulator

An interactive particle-based simulator that brings 6 seasons to life with dynamic effects and smooth transitions.

## 🎮 Seasons Included

| Season | Effect |
|--------|--------|
| ❄️ Winter | Snowflakes drifting |
| 🌸 Spring | Petals floating |
| ☀️ Summer | Heat waves shimmering |
| 🌧️ Monsoon | Rain streaks falling |
| 🍂 Autumn | Leaves swirling |
| ⛈️ Storm | Lightning striking |

## 🖱️ How to Use

- **Click** anywhere → Next season
- **Swipe** left/right → Change season  
- **Arrow buttons** → Navigate manually

## 🛠️ Built With

- HTML5 Canvas
- CSS3 Animations
- Vanilla JavaScript

## 📱 Responsive

Works on desktop, tablet, and mobile devices.
